Cipher App

Welcome to the 'Cipher App'. This is an application developed using Python (Flask). This app enables you to encrypt and decrypt data using two very basic ciphering techniques:


  • The Ceaser cipher
  • The Transposition cipher

The application is easy to use. Simply select one on the links at the top right, and paste/write the text you either want to encrypt or decrypt; the app process you text and return a block of either encrypted or decrypted text.
